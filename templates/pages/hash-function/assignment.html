Assignments

Practice Exercises

Exercise 1: Basic Hash Properties

Using the simulation tool, complete the following tasks:

  1. Generate MD5, SHA-1, and SHA-256 hashes for the text "Hello, World!"
  2. Compare the length of each hash output
  3. Document the time taken for each hash function
  4. Explain the differences you observe

Exercise 2: Avalanche Effect

Demonstrate the avalanche effect:

  1. Generate a SHA-256 hash for a sentence of your choice
  2. Change one character in your input
  3. Generate the new hash
  4. Compare the two hashes and calculate what percentage of bits changed
  5. Explain why this property is important for cryptographic hash functions

Exercise 3: Collision Resistance

Explore collision resistance:

  1. Generate MD5 hashes for 10 different inputs of your choice
  2. Try to find two inputs that produce similar hash patterns
  3. Explain why finding collisions is computationally difficult
  4. Research and discuss why MD5 is no longer considered cryptographically secure

Challenge Problems

Challenge 1: Hash Chain

Create a hash chain:

  1. Start with an initial value
  2. Create a chain of 5 hashes where each hash is the input for the next hash
  3. Document each step in the chain
  4. Explain potential applications of hash chains

Challenge 2: Password Security

Analyze password hashing:

  1. Create three different passwords of varying complexity
  2. Generate SHA-256 hashes for each password
  3. Analyze the relationship between password complexity and hash patterns
  4. Discuss why salting is important in password hashing

Submission Guidelines

Your submission should include:

  • Complete solutions to all exercises and challenges
  • Screenshots of your simulation results
  • Detailed explanations of your observations
  • Analysis of the results and conclusions drawn

Submit your work as a single PDF document named: HashFunction_Assignment_[YourName].pdf
