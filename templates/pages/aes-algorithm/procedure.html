Advanced Encryption Standard (AES) - Procedure

Overview

This procedure guide outlines the step-by-step process of implementing and using the AES algorithm. Follow these steps carefully to understand and work with AES encryption and decryption.

1. Preparation

  1. Understand the input requirements:
    • Plaintext must be exactly 128 bits (16 bytes)
    • Key size can be 128, 192, or 256 bits
    • Input can be in various formats (hex, ASCII)
  2. Prepare the state array:
    • Organize data in 4Ã—4 matrix of bytes
    • Fill in column-major order
    • Convert input format if necessary

2. Key Expansion

  1. Generate the key schedule:
    • Take the initial cipher key
    • Apply key expansion routine
    • Generate required number of round keys
  2. For each round key:
    • Apply RotWord operation
    • Apply SubWord operation
    • XOR with round constant (Rcon)

3. Encryption Process

  1. Initial round:
    • AddRoundKey with initial key
  2. Main rounds (9, 11, or 13 rounds):
    • SubBytes transformation
    • ShiftRows transformation
    • MixColumns transformation
    • AddRoundKey transformation
  3. Final round:
    • SubBytes transformation
    • ShiftRows transformation
    • AddRoundKey transformation

4. Decryption Process

  1. Use inverse operations:
    • InvSubBytes
    • InvShiftRows
    • InvMixColumns
  2. Apply round keys in reverse order:
    • Start with final round key
    • Work backwards through round keys

5. Implementation Steps

  1. Set up the environment:
    • Initialize necessary tables (S-box, etc.)
    • Prepare data structures
  2. Implement core functions:
    • Key expansion routine
    • Round transformations
    • Inverse transformations
  3. Add helper functions:
    • Data conversion routines
    • Matrix operations
    • Utility functions

6. Testing and Validation

  1. Use test vectors:
    • NIST test vectors
    • Known plaintext-ciphertext pairs
  2. Verify intermediate values:
    • Check state after each round
    • Validate round keys
  3. Test edge cases:
    • Different key sizes
    • Special input patterns
