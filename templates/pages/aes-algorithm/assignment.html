AES Algorithm - Assignments

Practice Exercises

Exercise 1: Basic AES Operations

Complete the following tasks using the simulation tool:

  1. Perform AES-128 encryption on a given plaintext block
  2. Document the state array after each round
  3. Perform decryption and verify the result
  4. Explain the changes in each round
  5. Compare the avalanche effect with DES

Exercise 2: Key Schedule Analysis

Analyze the key expansion process:

  1. Generate key schedules for different key sizes
  2. Document the round key generation process
  3. Compare the security implications of different key sizes
  4. Identify patterns in the expanded keys

Exercise 3: S-Box Properties

Study the AES S-box:

  1. Analyze the nonlinearity properties
  2. Calculate the difference distribution table
  3. Verify the bijective property
  4. Compare with other S-box designs

Challenge Problems

Challenge 1: Modes of Operation

Implement and analyze different modes:

  1. Implement ECB, CBC, CTR modes
  2. Compare security properties
  3. Analyze performance characteristics
  4. Discuss use cases for each mode
  5. Implement padding schemes

Challenge 2: Performance Analysis

Analyze AES performance:

  1. Measure encryption/decryption speeds
  2. Compare different key sizes
  3. Analyze memory usage
  4. Suggest optimization strategies

Challenge 3: Security Analysis

Study security aspects:

  1. Research known attacks on AES
  2. Analyze side-channel vulnerabilities
  3. Implement basic timing attacks
  4. Propose countermeasures

Implementation Tasks

Task 1: Basic Implementation

Create a complete AES implementation:

  1. Implement key expansion
  2. Implement all round transformations
  3. Add encryption/decryption functions
  4. Include test vectors
  5. Document the implementation

Task 2: Optimization

Optimize the implementation:

  1. Implement T-table approach
  2. Use lookup tables for efficiency
  3. Optimize memory usage
  4. Measure performance improvements

Task 3: Integration

Create a practical application:

  1. Build a file encryption tool
  2. Implement secure key management
  3. Add error handling
  4. Create a user interface

Submission Guidelines

Your submission should include:

  • Complete solutions to all exercises
  • Working code implementations
  • Analysis reports
  • Performance measurements
  • Security analysis documentation
