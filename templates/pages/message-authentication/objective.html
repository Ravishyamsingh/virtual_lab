Objectives

After completing this laboratory, you will be able to:

  • Understand the importance of message authentication in secure communications
  • Learn different types of message authentication techniques:
    • Message Authentication Codes (MACs)
    • Hash-based Message Authentication Codes (HMACs)
    • Digital Signatures
  • Implement and analyze various message authentication algorithms
  • Verify message integrity using different authentication methods
  • Understand the differences between various authentication techniques
  • Learn about common attacks on message authentication systems
