Assignments

Practice Exercises

Exercise 1: Basic DES Operations

Using the simulation tool, complete the following tasks:

  1. Perform DES encryption with a given key on a 64-bit plaintext block
  2. Track and document the state after each round
  3. Perform the decryption process
  4. Verify that you recover the original plaintext
  5. Document the complete process with intermediate values

Exercise 2: Key Schedule Analysis

Study the key schedule generation:

  1. Generate the complete key schedule for a given key
  2. Document the rotation patterns for each round
  3. Analyze the differences between consecutive round keys
  4. Explain how the key schedule contributes to security

Exercise 3: S-Box Analysis

Investigate S-box properties:

  1. Track how specific 6-bit inputs map to 4-bit outputs
  2. Analyze the nonlinearity of S-box transformations
  3. Document how S-boxes contribute to the avalanche effect
  4. Compare different S-boxes and their properties

Challenge Problems

Challenge 1: Avalanche Effect

Demonstrate and analyze the avalanche effect:

  1. Choose a plaintext and key
  2. Change one bit in the plaintext and observe changes
  3. Change one bit in the key and observe changes
  4. Calculate and compare the number of bits that change
  5. Analyze the propagation of changes through rounds

Challenge 2: Weak Keys

Study weak keys in DES:

  1. Research and identify known weak keys in DES
  2. Demonstrate the effect of weak keys
  3. Analyze the key schedule patterns for weak keys
  4. Propose methods to detect and avoid weak keys

Challenge 3: Implementation

Create a simplified implementation:

  1. Implement the Feistel network structure
  2. Create the key schedule generation
  3. Implement the round function
  4. Add the initial and final permutations
  5. Test with various inputs and verify results

Analysis Tasks

Security Analysis

Conduct a thorough security analysis:

  1. Analyze the strength of the algorithm against known attacks
  2. Discuss the implications of the 56-bit key size
  3. Compare DES with other block ciphers
  4. Propose improvements or alternatives

Performance Analysis

Evaluate performance aspects:

  1. Measure encryption/decryption times
  2. Analyze the impact of different input sizes
  3. Compare software vs. hardware implementation considerations
  4. Suggest optimization strategies

Submission Guidelines

Your submission should include:

  • Complete solutions to all exercises and challenges
  • Detailed analysis and observations
  • Screenshots of simulation results
  • Implementation code (if required)
  • Security and performance analysis reports

Submit your work as a single PDF document named: DES_Assignment_[YourName].pdf
